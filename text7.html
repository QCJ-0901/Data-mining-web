Practical Work Assignment 7

Overview:

Error analysis based on predicted and actual values, calculating MAE and MAPE metrics.

Task Requirements:

  1. Calculate absolute error
  2. Calculate Mean Absolute Error (MAE)
  3. Calculate Mean Absolute Percentage Error (MAPE)

MAE = (Σ|Y-Y'|)/n
MAPE = (Σ|(Y-Y')/Y|)/n × 100%

Example Calculation:
MAE = (2+1+3+0+1)/5 = 7/5 = 1.4
MAPE = (0.5+0.2+0.6+0+0.14)/5 × 100% = 28.8%
