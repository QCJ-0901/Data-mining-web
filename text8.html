Overview:

Outlier range calculation based on Interquartile Range (IQR).

Task Requirements:

  1. Calculate IQR value
  2. Determine outlier boundaries
  3. Define mathematical range

Given Parameters:
Q₁ = 12，Q₃ = 18

Calculation Process:
IQR = Q₃ - Q₁ = 6
Lower Bound = Q₁ - 1.5×IQR = 3
Upper Bound = Q₃ + 1.5×IQR = 27

Outlier Range:

Mathematical Expression:

(-∞, 3] ∪ [27, +∞)
