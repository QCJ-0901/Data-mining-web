📊 Core Data Mining Patterns

1. Descriptive and Predictive Patterns

  • Descriptive Mining: Characterizes general properties of data, including:
    • Data characterization (summarizing target class data)
    • Data discrimination (comparing target class with contrast class)
  • Predictive Mining: Makes predictions through induction, including:
    • Classification and regression analysis
    • Outlier detection

2. Class/Concept Characterization Techniques

Example: Analyzing high-value customer characteristics

  • Age distribution: 40-50 years old (65%)
  • Occupation: Executives (42%)
  • Purchase frequency: ≥5 times per month

4. Association Rule Metrics

  • Lift: Effectiveness measure of the rule
